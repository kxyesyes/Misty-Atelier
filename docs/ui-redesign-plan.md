# 🖼️ Misty Atelier - 高端前台 UI 视觉彻底重构计划
*(The "Invisible Museum" Editorial Redesign)*

## 0. 为什么现在的 UI “太丑”且不搭？
目前的 UI 偏向于常规的“博客/标准作品集”：方正的卡片、死板的网格、明显的边框。
而我们计划引入的：**WebGL 玻璃雨滴、四季全局溶解切换、微光粒子**，属于行业最高端、极具情绪流动的特效。如果特效是“流动的”，而 UI 是“方正死板”的，两者就会产生严重的割裂感。我们需要把网站改造成一个**Awwwards 级别的数字艺术馆**。

---

## 1. 核心设计理念：隐形的博物馆 (The Invisible Museum)
**UI 不应该是装画的“框”，而应该是画的“影子”。**
把一切多余的边框、背景色块全部拆掉。让排版像是一本高端的且排版考究的欧洲独立艺术杂志（Editorial Design）。

*   **极简主义与大留白**：大面积的留白，给动态天气和光影留出蔓延的空间。
*   **光学玻璃面板 (Glassmorphism)**：由于背景会有下雨、飘雪、四季渐变，所有的 UI 面板（如说明文字底色、导航栏）都应该采用 `backdrop-blur`（毛玻璃效果）。这样雨滴和飘云可以在文字底层透出来。
*   **无边框与反网格**：打破对齐的卡片，画作应该是“漂浮”在页面中的，而不是被塞进一个格子里。

---

## 2. 视觉锚点与排版系统重构

### 🖋️ Typography (排版体系的降维打击)
*   **主标题 (Serif)**：极具古典雕版气质的超大号 Serif（衬线体），比如 `Playfair Display` 或 `Bodoni`，带有非常优美的连字（Ligatures）。
*   **副文本与数据 (Sans-Serif)**：所有诸如“日期、尺寸、标签”等元数据，采用极小字号（10px-11px）、大字距（Letter-spacing: 0.2rem）的无衬线体（如 `Inter`），全部大写，展现绝对的现代与克制。

### 🖱️ 交互体验 (流畅与通感)
*   **贯穿全站的平滑滚动 (Smooth Scroll)**：引入 `Lenis` 或 `Locomotive Scroll`。普通的网页滚动是生硬的，艺术展级别的网页，滚动应当像黄油一样顺滑，带有轻微的惯性缓冲。
*   **定制鼠标指针 (Blend-mode Cursor)**：隐藏系统默认鼠标，使用一个跟随鼠标漂移的小圆点。当指针划过深色画作时变成白色，划过白色背景时变成黑色（`mix-blend-mode: difference`）。
*   **视差体验 (Parallax)**：当页面向下滚动时，背景的画作以极慢的速度往上偏移，文字以较快的速度飘过，产生空间的深度（配合雨天特效，三维感极强）。

## 3. 核心页面的彻底颠覆

### 首页 (Homepage) - 从“列表”到“沉浸式剧场”
*   弃用目前的块状布局。
*   一屏只展示一张画的绝美局部。背景铺满这张画带来的“雨水/雾气/四季光影”，画幅以不对称的比例悬浮在画面中央。
*   用户往下滚动，不是干瘪的卡片罗列，而是通过巨大的字体和“擦除效果（Reveal on scroll）”引导出下一张巨幅画作。

### 作品展览页 (The Exhibition Archive) - 从“格子”到“瀑布飘浮”
*   现在的九宫格太像电商网站。
*   改为 **非对称瀑布流 (Asymmetrical Masonry)** 或 **水平无限画廊 (Horizontal Horizontal Scroll)**。部分画作大，部分画作极小。它们在不同的 Z 轴高度上。滚动时，因为视差，前排的小画移动得快，后排的大画移动得慢。
*   画作不再有直白的文字。只有当定制的鼠标悬停在画作上时，排版精美的标题才会在鼠标旁边像游丝一样浮现出来。

### 艺术品详情页 (Artwork Detail) - 从“纯看画”到“阅读故事”
*   一进页面，画作极具冲击力地占据左侧或右侧的绝对 100vh 高度（半屏或全屏）。
*   另一侧是优雅的留白区域。当背景在下雨时，这边的留白甚至能听到雨声，文字像诗歌一样排布在留白处。页面不产生垂直位移，而是**画作黏性固定 (Sticky)**，文字内容缓慢地滚过。

## 4. 实施阶段拆解
为了平滑过渡现有的代码，我们分为三步：

1. **第一期（基础骨架清洗）**：
   - 引入平滑滚动引擎 (`@studio-freight/lenis`)。
   - 拆除现有项目里所有的 `.rounded-card`, `.ring-1`, `.bg-cream` 等具有物理隔离感的 CSS 盒子，彻底清空画布背景，准备迎取超大艺术字体系统和全屏天气特效。
2. **第二期（杂志级排版与视差重构）**：
   - 完全重写 ArtworkCard，实现不对称悬浮布局。
   - 加入全站的磁性鼠标指针（Custom Cursor）与视差（Parallax）指令。
3. **第三期（特效深空融合）**：
   - 将之前设计的 WebGL 玻璃雨、四季循环全盘接入。这时候的系统因为没有了“丑陋的盒子拘束”，雨水能顺着满屏的画作与杂志般的字体肆意流淌。
