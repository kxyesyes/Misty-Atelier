# 🌸 🌻 🍂 ❄️ Misty Atelier - 四季主题美学设计方案
*(The Aesthetics of Seasons: UI Temperature & Pacing)*

## 设计理念：超越“装饰”的“感知设计”
在传统的网页设计中，季节主题往往只停留在“换个背景色”或“加个飘落特效”。
但在 `Misty Atelier` 这样极具艺术感和静谧气质的项目中，四季不应当只是视觉修饰，而应当是**UI 的“温度 (Temperature)”与“节奏 (Pacing)”**。每个季节都应该让鼠标滑动、页面过渡、色彩和留白产生截然不同的“触觉”。

---

## 🌸 春：【呼吸与复苏】(Breath & Awakening)
**美学核心**：水汽感、萌发、柔软的过渡。
*   **空间色彩**：不再是纯粹的白，而是一种带有极淡青瓷色（Pale Celadon）和晨雾白（Morning Mist）的复合底色，UI 控件边缘的明暗对比降到最低，呈现水彩般的晕染感。
*   **UI 创新（呼吸式排版）**：
    *   **出场动画**：图片和文字出现时的 `Framer Motion` 动效采用“向外微微扩张（Scale 0.98 -> 1）”加“缓入（Ease-out）”的呼吸节奏，像植物展开叶片。
    *   **排版细节**：字距（Letter-spacing）比平时稍微放宽一点点，让文字“透出空气感”。
*   **微光特效**：屏幕角落或底层会有极其缓慢的、不规则的 SVG 色块晕染（类似水彩在宣纸上散开的模糊光晕。

## 🌻 夏：【光尘与长昼】(Dust & The Long Day)
**美学核心**：高对比的界限、光线、怠倦感（慵懒的午后）。
*   **空间色彩**：刺眼的亮纸色（Bleached Paper）作为受光面，配合你原有的极深墨色 `--color-ink` 作为重阴影（深邃的室内）。模拟“从刺眼的室外看入凉爽暗室”的体感。
*   **UI 创新（丁达尔与光尘）**：
    *   **丁达尔穿透光**：页面采用纯 CSS 的混合模式（`mix-blend-mode: soft-light`），在页面固定切出一个 45 度的隐形光栅。当用户滚动页面时，画作滑过这条光栅，会自动亮起一抹带有微小“浮尘粒子（Canvas 2D）”的高光。
    *   **滞后感滚动**：整体的页面滚动或 Hover 动效可以加入一点微妙的“阻尼感/滞后感”，模拟夏日午后时间被拉长的慵懒体感。

## 🍂 秋：【信笺与琥珀】(Letters & Amber)
**美学核心**：怀旧、纸张触感、温度、颗粒感。
*   **空间色彩**：整个 UI 切换为琥珀色（Amber）、枯叶褐、和粗糙的暖色羊皮纸底色。
*   **UI 创新（质感深化）**：
    *   **纸纹滤镜 (Paper Grain)**：加深你原本就有的 `.paper-grain` CSS 背景噪点，让整个屏幕看起来像是一封放久的信封。
    *   **Hover 陈旧效应**：当鼠标悬停在按钮或图片上时，滤镜会有一瞬间的 `sepia`（泛黄/褪色）效果闪过，接着才显示原本的鲜艳色彩，制造一种“拂去灰尘重新回想起这段记忆”隐喻。
*   **边缘剥落**：使用 CSS 的 Mask-image，让图片或卡片的四周呈现极轻微的不规则粗糙感（类似手工撕纸或拍立得相纸的毛边）。

## ❄️ 冬：【留白与霜边】(Whitespace & Frost)
**美学核心**：绝对的静谧、清冷、极致的克制。
*   **空间色彩**：退回到最极简的状态，以冷白（Cold White）和灰蓝（Rain/Mist）为主，几乎剔除所有的暖色调。
*   **UI 创新（结霜与收缩）**：
    *   **收缩式排版**：冬天是收敛的。画作周边的留白空间（Padding/Margin）被刻意放大，让每一张画在屏幕中显得更小、更孤独、更加珍贵（如同雪地中的小木屋）。
    *   **暗角与霜边屏 (Frost Vignette)**：使用 CSS 在整个视口的四周建立一圈非常非常微弱的白色/浅蓝色渐变遮罩（Inner Shadow）。这模拟了人们在深冬从室内往外看时，窗户边缘凝结的白霜。
    *   **冰脆的交互**：所有的 Hover 和点击特效（Transition duration）变得极快且干脆俐落（例如 0.1s），没有夏天的拖泥带水，展现“冷”的物理反馈。

## 如何将这些主题融入代码架构？
这套四季系统可以通过 Next.js + React 的 **CSS Variables (CSS 变量)** 完美落地。

1. **设置全局 `<SeasonThemeProvider>`**：在 HTML `<body>` 标签动态注入 `data-season="spring"` 等属性。
2. **重定义 CSS Tokens**：根据对应的 `data-season`，重写 `--background`, `--foreground`, `--transition-speed`, `--grain-intensity` 等变量。
3. **数据映射**：在 `src/data/exhibitions.ts` 或具体的画展里，可以为特定展览设定强制的季节主题，当用户进入特定氛围的展览，UI 将完全潜入那个季节。
