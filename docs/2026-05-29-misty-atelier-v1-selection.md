## 2. V1 入选草案原则

第一版建议入选 54 张作品，落在已确认的 40-60 张范围内。

筛选策略：

1. 优先使用根目录下命名完整、风格统一、画幅适合展示的主线作品。
2. 每个主展区保持 8-10 张，避免某个展区过强或过弱。
3. 从子目录中少量补入可以增强展区层次的作品。
4. 陶瓷、头像、眼睛特写、简约壁纸等暂不进入 V1 主展区，保留给后续扩展。
5. 赛博和瓷器类只少量进入或暂缓，避免第一版气质过散。

## 6. 暂缓进入 V1 的素材方向

这些素材不代表质量低，只是不建议第一版混入主线：

- `传统瓷器美学` 和 `新陶瓷美学`：建议后续做独立展区 `Porcelain Silence / 瓷白静物`。
- `头像`：更适合角色头像墙或单独商业展示页。
- `眼睛特写图`：更适合实验性细节研究展区。
- `简约壁纸图`、`手机壁纸`、`电脑壁纸`：更适合后续 Archive 或 Wallpaper Lab。
- 大量 `ChatGPT Image ...` 文件：建议人工看图后再命名和筛选。
- `赛博图`：第一版只保留 1 张观察气质，后续可独立成 `Neon Reverie / 霓虹梦游`。

## 7. 下一步人工确认方式

建议下一步按这个流程确认：

1. 先看六个展区封面是否成立。
2. 每个展区删掉 1-2 张气质不合的作品。
3. 从候选扫描表中补入更喜欢的替代作品。
4. 最终锁定 48-54 张。
5. 再进入图片重命名、压缩和 metadata 建立。
